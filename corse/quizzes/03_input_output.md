# 03 输入与输出：小测

1) 读取用户输入使用的函数是？
- A. scan()
- B. input()
- C. gets()
- D. readline()

答案：B。

2) 格式化输出不属于 Python 语法的是？
- A. f"{name}"
- B. "{}".format(name)
- C. "%s" % name
- D. `$name`

答案：D。美元插值不是 Python 语法。

3) 将 `"42"` 安全转换为整数的异常类型是？
- A. TypeError
- B. ValueError
- C. KeyError
- D. IndexError

答案：B。转换失败会抛出 ValueError。

4) `print("a", "b", sep="-")` 输出？
- A. a b
- B. a-b
- C. ab
- D. a--b

答案：B。

5) `input()` 的返回类型是？
- A. int
- B. float
- C. str
- D. Any

答案：C。
