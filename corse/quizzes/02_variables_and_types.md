# 02 变量与类型：小测

1) `type(3.0)` 的结果是？
- A. <class 'int'>
- B. <class 'float'>
- C. <class 'double'>
- D. <class 'number'>

答案：B。

2) 下列关于动态类型的说法正确的是：
- A. 变量一旦定义，类型不可变
- B. 变量可以在运行期绑定不同类型的值
- C. Python 是静态类型语言
- D. 必须显式标注类型

答案：B。

3) `bool(0)` 与 `bool("")` 的结果分别为？
- A. True, True
- B. False, True
- C. True, False
- D. False, False

答案：D。0 与空字符串都视为 False。

4) 将字符串 `"123"` 转为整数使用？
- A. str("123")
- B. float("123")
- C. int("123")
- D. bool("123")

答案：C。

5) `isinstance(True, int)` 的结果？
- A. True
- B. False

答案：A。布尔是整型子类。
